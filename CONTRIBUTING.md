## How to contribute

First off, thank you for considering contributing to Albly

## Areas where you can help
* Wherever you see a problem or want to improve something

## How Can I Contribute?

### Contribution Workflow
1. Fork the repository
2. Create a new branch for your feature or bugfix
3. Commit your changes with a descriptive message
4. Push your branch
5. Open a Pull Request

### Reporting Bugs
* Check the [Issues tab](https://github.com/m4ykey/Albly/issues) to see if the bug has already been reported.
* If not, open a new issue. Describe the problem.

### Suggesting Enhancements
* I'm always looking for new features! Open an issue with the "enhancement" to discuss your ideas.
